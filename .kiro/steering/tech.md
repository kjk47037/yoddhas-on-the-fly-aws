# Tech Stack

## Frontend
- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS + DaisyUI components
- **State Management**: React useState/useEffect hooks
- **Routing**: React Router DOM
- **Charts**: Recharts for data visualization
- **Icons**: React Icons, FontAwesome

## Backend
- **Node.js Server**: Express.js (port 5005)
- **Python Server**: Flask (port 5004) for Twitter/Instagram APIs
- **ML Server**: Flask (port 5007) for ML predictions

## AI/ML Services
- **Text Generation**: HuggingFace Inference API (Llama 3.1)
- **Image Generation**: FLUX.1-schnell via HuggingFace Router
- **Video Generation**: ModelsLab text2video API
- **Competitor Analysis**: Google Gemini API
- **Custom ML**: scikit-learn RandomForest for engagement prediction

## Database & Auth
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth (email/password + social)
- **File Storage**: Local temp storage for generated images

## External APIs
- **Twitter API**: v2 for analytics and posting
- **Instagram API**: RapidAPI for competitor analysis
- **Twilio**: WhatsApp notifications

## Development Tools
- **Package Manager**: npm
- **Linting**: ESLint
- **Build Tool**: Vite
- **Python Environment**: venv with requirements.txt
